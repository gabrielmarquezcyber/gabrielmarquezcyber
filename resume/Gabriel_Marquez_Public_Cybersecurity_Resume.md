### Empire Breacher - AI/Web3 Wallet-Agent Security Research Harness

Repository: https://github.com/gabrielmarquezcyber/empire-breacher

Built a controlled AI/Web3 security research harness modeling prompt-injection and authority-boundary risk in wallet-enabled agents. The project separates content classification from behavior enforcement and validates safe behavior using deterministic tests.

**Key work:**

* Modeled prompt-injection risk in wallet-enabled AI agents
* Created malicious, ambiguous, and benign test fixtures
* Built a toy wallet-agent simulator
* Implemented deterministic PASS / REVIEW / FAIL evaluation
* Mapped behavior outcomes:

  * PASS -> summarize_safely
  * REVIEW -> request_human_review
  * FAIL -> refuse_wallet_action
* Enforced the invariant that external content can be analyzed but cannot authorize wallet behavior
* Added Python regression tests, JSON decision outputs, safety policy documentation, public-safe validation reports, and proof map

**Relevant roles:** AI Security, AppSec, Product Security, Web3 Security, LLM/agent security, prompt injection, automation security, and forward-looking security operations roles involving AI-assisted workflows.
